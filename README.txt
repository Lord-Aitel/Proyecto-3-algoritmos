Descripción
Este código implementa un programa que realiza carreras entre diferentes algoritmos de ordenamiento utilizando conjuntos de datos específicos. Los conjuntos de datos incluyen colas de espera, trazabilidad de objetos y eventos en diferentes escenarios. Los algoritmos de ordenamiento incluidos en la carrera son:

Bubble Sort
Heap Sort
Quick Sort
Merge Sort
Selection Sort
Insertion Sort
Shell Sort
El programa permite al usuario seleccionar el tipo de conjunto de datos (colas de espera, trazabilidad de objetos o eventos) y ejecutar la carrera entre los algoritmos de ordenamiento para evaluar su rendimiento.

Uso
El programa se inicia con la función main() y presenta un menú de opciones para que el usuario elija el tipo de carrera que desea realizar. Las opciones incluyen:

Colas de Espera: Carrera de algoritmos con conjuntos de datos de colas de espera.
Trazabilidad de Objetos: Carrera de algoritmos con conjuntos de datos de trazabilidad de objetos.
Eventos de Cada Escenario: Carrera de algoritmos con conjuntos de datos de eventos en diferentes escenarios.
Salir: Salir del programa.
El usuario puede seleccionar la opción deseada ingresando el número correspondiente. Después de cada carrera, se muestran los resultados, incluido el algoritmo más rápido.

Conjuntos de Datos
El código incluye conjuntos de datos predefinidos para colas de espera, trazabilidad de objetos y eventos en diferentes escenarios. Estos conjuntos se generan aleatoriamente y se utilizan para evaluar el rendimiento de los algoritmos de ordenamiento.

Personalización
El código permite la personalización de parámetros como la presencia de duplicados, el orden de los datos y la aleatoriedad en la generación de conjuntos de datos.

Compilación
El código está escrito en C++ y puede compilarse utilizando un compilador estándar de C++. Asegúrese de incluir las bibliotecas necesarias al compilar.

Ejecución
Ejecute el programa compilado y siga las instrucciones en el menú para realizar carreras entre los algoritmos de ordenamiento.
